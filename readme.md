# Hermione-m

Simple app for spaced repetitions

## Commands

| Command           | Action                                          |
|----------------   |-------------------------------------------------|
| `npm install`     | install dependencies                            |
| `npm run dev`     | start development mode                          |
| `npm run build`   | build project                                   |
| `npm run preview` | run preview                                     |
| `npm run lint`    | lint code                                       |

## Info

- rendered by [react](https://reactjs.org/)
- controlled by [effector](https://effector.dev/)
- styled with [mantine](https://mantine.dev/)
- built via [vite](https://vitejs.dev/)
